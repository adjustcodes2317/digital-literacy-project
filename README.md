## About This Project

This repository is my Digital Literacy Portfolio submitted as part of the CSE0001 course at VIT Bhopal. The project is based on the role of a Student Digital Ambassador — someone who helps peers navigate the digital world responsibly. It covers five modules — digital tools, online portfolio building, coding platforms, professional communication, and cybercrime awareness.

## Task Summaries

### Task 1 — Digital Literacy Infographic
Created a one-page infographic using a design tool covering four topics — what digital literacy means, useful digital tools for students, safe internet practices, and professional email etiquette. The infographic is designed to be shared with batchmates as an awareness resource.

### Task 2 — Student Digital Portfolio
Set up accounts on GitHub, LinkedIn, and Kaggle. Created a GitHub profile README with name, branch, year, and learning goals. Updated LinkedIn with current education details. Screenshots of all profiles are in the task-2-portfolio folder.

### Task 3 — Coding and Collaboration Platforms
Completed a beginner coding challenge on HackerRank. Built a 5-question Digital Literacy Awareness Quiz using Google Forms with both multiple choice and short answer questions. The form link is shared below.

**Google Form Link:** https://docs.google.com/forms/d/e/1FAIpQLSfI1s5R7O8gA9xGuoKP7DyxvSmQDh8kUnznksA7YYYIj5lCuw/viewform?usp=publish-editor

### Task 4 — Professional Email Etiquette
Wrote two professional emails — one requesting a deadline extension from a professor and one expressing interest in a summer internship. Also created a Social Media Do's and Don'ts checklist for college students.

### Task 5 — Cybercrime Awareness
Wrote a case study on phishing attacks — how they work, who they target, and what the consequences are. Created a Stay Safe Online prevention checklist with 10 actionable tips including two specific to UPI and financial safety.
